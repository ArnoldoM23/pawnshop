Pawnshop is a single page website.

Stack:
Node.js, Express.js, React.js, Redux.js, Webpack

Team

Arnoldo Munoz 

Local Development

From within the root directory:

Step 1: Install dependencies
```
$ npm install
```
Step 2: Install Webpack
```
$ npm install -g webpack
```
Step 3: Run server
```
$ npm start
```
Step 4: Build project
```
$ npm run build
```
